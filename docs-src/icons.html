FontAwesome integration

Font installation

  • Download latest release from Official website
  • Unpack archive
  • Copy /fonts folder to your assets directory
It is possible to change the location of font files by defining $font-path variable (/fonts by default).

You can also use CDN url: 

$font-path: "//netdna.bootstrapcdn.com/font-awesome/4.3.0/fonts";
$fa-version: "4.0.0"; // Optional

Usage

FontAwesome integration is not included by default. There are two ways of add it:

Semantic
@import 'fa-icons';

.my-success-button {
  @include icon-before(success, 16px);
}

.my-check-button {
  @include icon-after(check);
}

li {
  &::before {
    @include icon-pseudo(circle);
  }
}
Non-semantic
<link rel="stylesheet" href="/css/fa-icons-classes.css">
<a class="icon before link" href="https://google.com/">Google</a>
