Customizing inputs

Here is default values for forms module:

$ae-input-color-normal:  #b7b7b7 !default;
$ae-input-color-valid:   #6dab5e !default;
$ae-input-color-invalid: #db4f4f !default;

Input module have a mixin for colorizing: 

input[type="text"] .mycolor {
  @include input-style(#f200ff);
}
